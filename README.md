# 🎺 管樂團樂器資產管理系統 (Instrument Asset Management System)

## 📖 專案簡介
為了解決傳統管樂團在樂器管理上，常面臨紙本登記易遺失、狀態追蹤困難（如忘記哪些樂器正在送修或被借出）等痛點，所設計的桌面端應用程式。

## 📖 系統操作教學 
### 1. 啟動與資料庫初始化
* 若為首次使用，系統會在背景自動建立標準格式的空資料庫。
* 畫面上方會即時更新並顯示目前的資產統計數據（在庫、借出、維修中）。
<img width="554" height="283" alt="image" src="https://github.com/user-attachments/assets/cfe57e81-6024-4871-a812-929d55118652" />


### 2. 新增樂器
* 點擊右下角 **「新增樂器」** 按鈕開啟建檔視窗。
* 依序填入資料：
  * **樂器編號：** 具備格式防呆，僅限輸入英文字母、數字與橫槓（例：`W-001`），防止輸入無效字元。
  * **樂器分類：** 透過下拉式選單選擇（如木管、銅管），統一資料格式。
  * **樂器名稱：** 填入具體名稱（如長笛、小號）。
* 點擊 **「確認新增」** 系統將自動設定成「在庫」狀態。
<img width="289" height="261" alt="image" src="https://github.com/user-attachments/assets/49cae3a7-1db2-4fd1-9666-ff6280bfb096" />


### 3. 狀態管理 (借出 / 歸還 / 送修)
請先於中央表格點選欲操作的樂器列，再點擊下方對應按鈕：
* **借出登記：** 僅限狀態為「在庫」的樂器可執行。若樂器已借出或維修中，系統會跳出阻擋提示。
* **送修登記：** 將樂器狀態變更為「維修中」，此狀態下的樂器將被系統鎖定，無法執行借出操作。
* **歸還登記：** 樂器使用完畢或保養結束後點擊此按鈕，狀態將安全恢復為「在庫」。

### 4. 關鍵字即時搜尋
* 於畫面上方 **「關鍵字搜尋」** 輸入框內直接打字。
* **無需按下 Enter 或搜尋鍵**
* 將搜尋框清空，即可恢復顯示所有樂器。
